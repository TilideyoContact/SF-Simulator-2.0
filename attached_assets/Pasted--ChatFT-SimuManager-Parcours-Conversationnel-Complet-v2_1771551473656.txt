**Logique de recommandation :** `getRecommandation(store)` → retourne le scénario le plus pertinent

```typescript
function getRecommandation(state: ParcoursState): string {
  // Priorité 1 : difficulté "feedback" ou "conduite" → feedback_recadrage
  if (state.difficulte === 'feedback' || state.difficulte === 'conduite')
    return 'feedback_recadrage';
  // Priorité 2 : QVT difficile ou engagement faible → decision_difficile
  if (state.barometre?.qvt === 'difficile' || state.barometre?.engagement === 'faible')
    return 'decision_difficile';
  // Priorité 3 : objectif "confiant" ou expérience débutant → feedback_positif
  if (state.objectifs?.includes('confiant') || state.experience === 'debutant')
    return 'feedback_positif';
  // Défaut
  return 'feedback_recadrage';
}
```

**Logique de calcul du niveau de difficulté :**

```typescript
function calculateDifficulty(persona: PersonaState): DifficultyLevel {
  let score = 0;

  // Profil DISC
  const discScores = { stable: 0, influent: 1, consciencieux: 2, dominant: 3 };
  score += discScores[persona.disc];

  // Relation
  const relationScores = { bonne: 0, neutre: 1, tendue: 3 };
  score += relationScores[persona.relation];

  // État d'esprit
  const etatScores = { positif: 0, neutre: 1, stresse: 2, agace: 3 };
  score += etatScores[persona.etatEsprit];

  // Type collaborateur (manager = +1 difficulté)
  if (persona.typeCollab === 'manager') score += 1;

  // Mapping score → niveau
  if (score <= 2) return 'facile';      // ⭐
  if (score <= 5) return 'modere';      // ⭐⭐
  if (score <= 8) return 'difficile';   // ⭐⭐⭐
  return 'expert';                       // ⭐⭐⭐⭐
}
```

**Matrice de référence (exemples) :**

| Combinaison | Score | Difficulté |
|-------------|-------|-----------|
| Stable + Bonne + Positif + Agent | 0 | ⭐ Facile |
| Influent + Neutre + Neutre + Agent | 3 | ⭐⭐ Modéré |
| Consciencieux + Tendue + Stressé + Agent | 7 | ⭐⭐⭐ Difficile |
| Dominant + Tendue + Agacé + Manager | 10 | ⭐⭐⭐⭐ Expert |

**Descriptions comportementales automatiques (exemples) :**

| DISC | Relation | État | Description |
|------|----------|------|-------------|
| Stable | Bonne | Positif | « un collaborateur coopératif et à l'écoute, qui cherchera des solutions avec vous » |
| Dominant | Tendue | Agacé | « un collaborateur direct et impatient, qui challengera vos arguments et demandera des actions concrètes rapidement » |
| Consciencieux | Neutre | Stressé | « un collaborateur méthodique mais sous pression, qui demandera des preuves factuelles avant d'adhérer » |
| Influent | Tendue | Neutre | « un collaborateur expressif mais méfiant, qui pourra digresser ou réagir émotionnellement » |

> **Vous jouez le rôle du manager. L'IA joue le rôle du collaborateur.**
>
> Écrivez votre première intervention comme si vous commenciez l'entretien. L'échange durera environ **[7-10]** tours.
>
> *À tout moment, tapez « /fin » pour terminer la simulation.* »

### ÉTAPE 16 — Mise en contexte + 1er message collaborateur IA

**Injection dans le prompt système :**

```
CONTEXTE SIMULATION :
- Scénario : [feedback_recadrage | feedback_positif | decision_difficile]
- Type collaborateur : [agent | manager]
- Profil DISC : [dominant | influent | stable | consciencieux]
- Relation préexistante : [bonne | neutre | tendue]
- État d'esprit initial : [positif | neutre | stresse | agace]
- Niveau de difficulté : [facile | modere | difficile | expert]
- Profil du manager (user) : [MP | MI | MS], expérience [X]
- Baromètre social : QVT [bonne|moyenne|difficile], engagement [bon|moyen|faible]

COMPORTEMENT IA SELON DISC :
[Dominant] → Réponses courtes et directes. Challenge. "Venez-en au fait."
[Influent] → Réponses longues, digressives. Réactions émotionnelles visibles.
[Stable] → Réponses mesurées, acceptation lente. Intériorise. Résistance passive.
[Consciencieux] → Demande des détails, des dates, des preuves. Analyse tout.

MODULATION PAR RELATION :
[Bonne] → Ouverture +2 tours, seuil de tolérance élevé.
[Neutre] → Comportement standard.
[Tendue] → Méfiance initiale, interprétation négative, résistance +2 tours.

MODULATION PAR ÉTAT D'ESPRIT :
[Positif] → Écoute active immédiate.
[Neutre] → Standard.
[Stressé] → Distrait, impatient, seuil bas.
[Agacé] → Résistance d'entrée, ton sec, fermeture rapide.

MODULATION PAR TYPE COLLABORATEUR :
[Agent] → Vocabulaire terrain, réactions émotionnelles directes, résistance émotionnelle.
[Manager] → Vocabulaire stratégique, contre-argumentation structurée, résistance intellectuelle.
```

**🤖 1er message du collaborateur IA (contextuel) :**

Exemple scénario feedback_recadrage + Agent + Stable + Neutre + Neutre :
> « *[L'agent entre dans le bureau et s'assoit calmement.]*
>
> Bonjour. Vous m'avez demandé de passer vous voir, j'espère que ce ne sera pas trop long, j'ai beaucoup de dossiers en attente. Je vous écoute. »

Exemple feedback_recadrage + Manager + Dominant + Tendue + Agacé :
> « *[Le manager entre, pose ses documents sur la table sans s'asseoir.]*
>
> Bon, vous vouliez me voir. Je vous préviens, j'ai un comité dans 30 minutes. De quoi s'agit-il ? »

### ÉTAPE 17 — Tours de jeu de rôle

**Nombre de tours selon scénario × type collaborateur :**

| Scénario | Agent | Manager |
|----------|:-----:|:-------:|
| Feedback / Recadrage | 7 tours | 8-9 tours |
| Feedback positif | 5-6 tours | 6-7 tours |
| Décision difficile | 7-8 tours | 9-10 tours |

**Chaque tour :**
1. Le manager (user) écrit son message
2. L'IA analyse (DISC + relation + état + scénario) et répond en personnage
3. L'IA ajuste son état interne : ouverture ↑ si bonne posture manager, fermeture ↑ si mauvaise posture

**Indicateurs internes IA (non visibles user) :**
- `ouverture` : 0-10 (évolue tour par tour)
- `positionDeVie` : OK+/OK+ → OK-/OK+ → OK-/OK- (évolution dynamique)
- `tournantDetecte` : boolean (moment clé où l'entretien bascule)
